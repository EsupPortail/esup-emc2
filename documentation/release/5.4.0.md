# Version 5.4.0 (XX/03/2025)

## Changements 

[CORRECTION]
* La liste des agents remontés pour les entretiens professionnels dans les structures considère maintenant la période en la date de prise de poste et la date de fin de campagne.
* Ajout d'un bandeau sur les écrans liés aux fiches de poste

### Modification des données

```postgresql
-- forçage de la date de convocation pour bricoler.
update entretienprofessionnel set convocation = date_entretien where convocation IS NULL;

-- Paramètre pour condition l'usage du bandeau sur les écrans présentant les fiches de poste
INSERT INTO unicaen_parametre_parametre(CATEGORIE_ID, CODE, LIBELLE, DESCRIPTION, VALEURS_POSSIBLES, ORDRE)
WITH d(CODE, LIBELLE, DESCRIPTION, VALEURS_POSSIBLES, ORDRE) AS (
    SELECT 'DISPLAY_BANDEAU_FICHEPOSTE', 'Affichage du bandeau sur les écrans présentant des fiches de poste', '<p>Active/désactive le bandeau présenter sur les écrans liés aux fiches de postes (associé au template <code>FICHEPOSTE_BANDEAU</code>)</p>', 'Boolean', 1000 
)
SELECT cp.id, d.CODE, d.LIBELLE, d.DESCRIPTION, d.VALEURS_POSSIBLES,  d.ORDRE
FROM d
JOIN unicaen_parametre_categorie cp ON cp.CODE = 'FICHE_POSTE';
update unicaen_parametre_parametre set valeur=true where code = 'DISPLAY_BANDEAU_FICHEPOSTE';

-- Nouvelles macros pour les affectations, échelons, grades et statuts à la date de situation de l'entretien
INSERT INTO unicaen_renderer_macro (code, description, variable_name, methode_name) VALUES
('ENTRETIEN#AgentAffectations','Retourne les affectations actives de l''Agent·e', 'entretien', 'toStringAgentAffectations'),
('ENTRETIEN#AgentEchelons','Retourne les échelons actifs de l''Agent·e', 'entretien', 'toStringAgentEchelons'),
('ENTRETIEN#AgentGrades','Retourne les grades actifs de l''Agent·e', 'entretien', 'toStringAgentGrades'),
('ENTRETIEN#AgentStatuts','Retourne les statuts actifs de l''Agent·e', 'entretien', 'toStringAgentStatuts'),
('ENTRETIEN#ResponsableAffectations','Retourne les affectations actives du Responsable', 'entretien', 'toStringResponsableAffectations'),
('ENTRETIEN#ResponsableEchelons','Retourne les échelons actifs du Responsable', 'entretien', 'toStringResponsableEchelons'),
('ENTRETIEN#ResponsableGrades','Retourne les grades actifs du Responsable', 'entretien', 'toStringResponsableGrades'),
('ENTRETIEN#ResponsableStatuts','Retourne les statuts actifs du Responsable', 'entretien', 'toStringResponsableStatuts');

-- Adaptations/corrections des privilèges
update unicaen_privilege_categorie set namespace='Agent\Provider\Privilege' where code='agent';
```

## Évolution des templates et macros

Nouveau template `FICHEPOSTE_BANDEAU` pour le message sur les écrans des fiches de poste. 
Il est à noter que si le template n'existe pas les bandeaux sont désactivé.

Nouvelle macro pour les affectations, grades, statuts et échelon des Agent·es et Responsable àla date de situation de l'entretien professionnelle 
- `ENTRETIEN#AgentAffectations`
- `ENTRETIEN#AgentEchelons`
- `ENTRETIEN#AgentGrades`
- `ENTRETIEN#AgentStatuts`
- `ENTRETIEN#ResponsableAffectations`
- `ENTRETIEN#ResponsableEchelons`
- `ENTRETIEN#ResponsableGrades`
- `ENTRETIEN#ResponsableStatuts`


## Évolution des paramètres

Nouveau paramètre `DISPLAY_BANDEAU_FICHEPOSTE` dans la partie fiches de poste pour activer/désactiver le bandeau associé au message informatif à propos des fiches de poste.
